# CSharp-Visual-Programming-WinForms

This repository contains my C# Windows Forms projects completed during my coursework. Each project demonstrates different aspects of Windows Forms development, object-oriented programming, file I/O, and interactive UI design. Every project is maintained in its own branch.

---

## Projects

### [Favorite Multi-Page Application](https://github.com/excellpepple/CSharp-Visual-Programing-WinForms/tree/favorite-multi-page-application)
- **Overview:** A multi-page image selection form displaying categories across pages and responding to user clicks.
- **Key Concepts:** Pagination, dynamic UI updates, event-driven programming, object naming conventions.
- **Skills Demonstrated:** Windows Forms, control visibility, modular code design.

### [Store Checkout](https://github.com/excellpepple/CSharp-Visual-Programing-WinForms/tree/Store-checkout)
- **Overview:** A kiosk-style application for adding items to a cart, calculating totals, tax, and optional tips.
- **Key Concepts:** Variables, constants, arithmetic operations, event handling, currency formatting.
- **Skills Demonstrated:** UI design, state management, exception handling, incremental calculations.

### [Password Checker](https://github.com/excellpepple/CSharp-Visual-Programing-WinForms/tree/Password-Checker)
- **Overview:** Validates passwords against length and character rules; includes a password generator.
- **Key Concepts:** Loops, conditionals, ASCII handling, flag-based validation, random generation.
- **Skills Demonstrated:** String parsing, logic design, listbox usage, input validation, modular methods.

### [Multi-Form Store](https://github.com/excellpepple/CSharp-Visual-Programing-WinForms/tree/multi-form-store)
- **Overview:** Shopping cart application spanning multiple forms with a dynamic cart on the main form.
- **Key Concepts:** Multi-form architecture, method modularization, menu-driven navigation, dynamic calculations.
- **Skills Demonstrated:** Object-oriented design, UI coordination, cart management.

### [Movie Collection](https://github.com/excellpepple/CSharp-Visual-Programing-WinForms/tree/Movie-Collection)
- **Overview:** Manage a personal movie collection, add and read movies from files, and navigate entries.
- **Key Concepts:** Parallel arrays, file I/O, multiple forms, combo boxes, radio buttons, checkboxes.
- **Skills Demonstrated:** Data persistence, UI updates from arrays, file dialogs, modular methods.

### [GPA Calculator (Final Project)](https://github.com/excellpepple/CSharp-Visual-Programing-WinForms/tree/GPA-Calculator)
- **Overview:** Track semesters, courses, and GPA. Supports adding, editing, removing classes, and saving/loading semester data.
- **Key Concepts:** Classes, enums, dictionaries, methods (void & value-returning), file I/O, validation, multi-form interaction.
- **Skills Demonstrated:** Object-oriented design, data modeling, GPA logic, file persistence, maintainable code.
